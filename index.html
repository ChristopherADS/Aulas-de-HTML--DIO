Treinamento Pratico HTML

(Voltar)

A escola Digital Innovation One (DIO) oferece treinamento prático em HTML por meio de seus cursos online.

Em geral, esses cursos incluem aulas em vídeo ministradas por instrutores experientes na área, bem como exercícios práticos

que os alunos podem fazer para reforçar o aprendizado.

No treinamento, os alunos aprendem os conceitos básicos do HTML, incluindo sua estrutura, tags e atributos.

Como funciona as TAGS

(Voltar)

Tags em HTML são usadas para definir elementos dentro de uma página da web e fornecer ao navegador instruções

sobre como renderizar o conteúdo da página. As tags são geralmente compostas de uma palavra-chave ou nome de tag,

cercada por colchetes angulares (< >).

Cada tag tem uma função específica e pode ser usada para criar vários tipos de elementos, como texto, imagens, links, formulários e tabelas.

Algumas tags são usadas para definir a estrutura básica da página, enquanto outras são usadas para estilizar o conteúdo da página.

Antes de utilizar usar as TAGS, utilizar (< >)

html: define o início e o final do documento HTML.

head: define informações sobre o documento, como o título e meta informações.

body: define o conteúdo da página, como texto, imagens e outros elementos.

p: define um parágrafo de texto.

img: define uma imagem.

a: define um link.

form: define um formulário.

table: define uma tabela.

Além disso, as tags podem ter atributos, que fornecem informações adicionais sobre o elemento.

Os atributos são especificados dentro da tag e geralmente incluem um nome e um valor, separados por um sinal de igual (=).

Por exemplo, a tag pode ter um atributo src para especificar a URL da imagem a ser exibida.

Em resumo, as tags em HTML são usadas para definir a estrutura e o conteúdo de uma página da web,

fornecendo instruções para o navegador sobre como renderizar o conteúdo da página.

Atributos

(Voltar)

Atributos em HTML são usados para fornecer informações adicionais sobre um elemento HTML. Eles são usados para definir

características específicas do elemento, como sua cor, tamanho, URL, nome, valor, etc. Os atributos são adicionados dentro da tag HTML,

após o nome da tag, usando o sinal de igual (=) e o valor do atributo, entre aspas duplas.

Outros exemplos de atributos comuns incluem o atributo "href" usado em tags para definir o destino do link, o atributo "class"

usado para definir uma classe CSS para o elemento, o atributo "id" usado para definir um identificador único para o elemento,

entre outros.

Os atributos em HTML são usados para fornecer informações adicionais sobre elementos HTML, permitindo que sejam estilizados e interativos.
